# AssetFlow MVP - Asset Management System

## Environment Variables
- `GEMINI_API_KEY`: Required for AI features (if any).
- `JWT_SECRET`: Secret key for JWT authentication.
- `PORT`: 3000 (Fixed).

## Database
Uses SQLite (`database.sqlite`) for local development.

## Roles
1. `admin` - Full access.
2. `maintenance` - Task management and asset condition updates.
3. `custodian` - Asset requests and reporting.
